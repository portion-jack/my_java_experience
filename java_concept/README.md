# JAVA

> ## Java
> JDK(Java Development Kit) > JRE(Java Runtime Environment) > JVM(Java Virtual Machine)

> ## Java
> 일반적인 순서
> 1. .java file 생성
> 2. .class 바이트 코드파일 생성
> 3. JVM 메모리 할당
> 4. main() 메서드 실행

> ## Java의 메모리
```
1. 클래스 영역 (정적 영역, 상수 영역, 메서드 영역)
2. 스택 영역 (변수 저장)
3. 힙 영역 (객체 저장)


기본 자료형(소문자로 시작)
    - boolean
    - byte, short, int, long
    - float, double
    - char

참조 자료형(대문자로 시작)
    - 객체 (배열, 클래스, 인터페이스)
```
